## SeDivider
### Name 
`se-divider`
### Props
| Name       | Default | Type    | Description                                                         |
| ---------- | ------- | ------- | ------------------------------------------------------------------- |
| horizontal | `false` | Boolean | A divider can segment content horizontally                          |
| inverted   | `false` | Boolean | A divider can have its colors inverted                              |
| fitted     | `false` | Boolean | A divider can be fitted, without any space above or below it.       |
| hidden     | `false` | Boolean | A hidden divider divides content without creating a dividing line   |
| section    | `false` | Boolean | A divider can provide greater margins to divide sections of content |
| clearing   | `false` | Boolean | A divider can clear the contents above it                           |
